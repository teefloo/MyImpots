# MyImpots 🇫🇷

**MyImpots** est une plateforme moderne et intuitive conçue pour simplifier la gestion de votre fiscalité personnelle en France. Que vous soyez salarié, indépendant ou retraité, MyImpots vous accompagne dans la compréhension et l'optimisation de votre déclaration de revenus.

## 🚀 Fonctionnalités

- **Calculateurs & Simulateurs**:
  - Simulateur d'impôt sur le revenu.
  - Arbitrage Micro-foncier vs Réel.
  - Calcul des frais réels et crédits d'impôt.
- **Explorateur de Cases**: Détails complets sur chaque case de la déclaration (2042, etc.).
- **Calendrier Fiscal**: Ne manquez aucune échéance grâce au suivi personnalisé.
- **Gestion Documentaire**: Checklist des documents nécessaires pour votre déclaration.
- **FAQ Fiscalité**: Réponses claires aux questions les plus fréquentes.
- **Formulaires**: Accès rapide aux formulaires officiels (CERFA).

## 🛠 Tech Stack

- **Framework**: [Next.js 15](https://nextjs.org/) (App Router)
- **Engine**: [React 19](https://react.dev/)
- **State Management**: React Hooks & Context
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Analytics**: [Vercel Analytics](https://vercel.com/analytics)
